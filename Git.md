# git 的一些常用命令

##### 1，初始化仓库

`git init` 在文件夹里生成.git文件夹

##### 2、  将修改提交到暂存区

`git add <filename>`

或者用`git add .`全部提交。

##### 3、将暂存区提交到版本库

`git commit -m ""`在引号里面添加的是对此次修改的描述

 ##### 4、提交到远程库

`git push origin master` 将主线提交到远程库

##### 5、从远程库克隆

`git clone <网址>` 网址就是github上面的地址

##### 6、查看历史提交

`git log --oneline`

##### 7、查看commit历史

`git reflog`
